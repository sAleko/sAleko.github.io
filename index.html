About me

Passionate about exploring the limitless possibilities of technology, I thrive on creating and enhancing virtual environments to make a meaningful impact.

Exploring Technology, Building Virtual Worlds

Throughout my education and work experience, I've learned many skills through making 3D games and experiences for VR (Virtual Reality) and non-VR platforms, including android and web-based experiences. Some of the skills I learned are programming and scripting, while also gaining some experience with 3D modeling and animation through Blender and Unity. I implemented the backend side, like network connections, databases, event handling, user input handling, and physics. I handled frontend elements as well, gaining great experience in creating and adjusting UX/UI elements. My passions for this work began in high school, when I learned how to use Unity and make my first game. I also was able to be an intern for a startup company called Take-Pause, where I was able to learn more about the workplace, and be able to experiment and learn more about Unity. In this internship, I created a VR world that is still used today to calm patients. My experience at Take-Pause started as an internship, then developed into a full time role, where I'm continuing to learn on the job. Take-Pause is where I learned how to develop in a team, where we use version control through GitHub, and create and maintain documentation.
